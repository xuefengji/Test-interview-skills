### 面试类型的分类

+ 技术面试（一面）
  + 广博的技术面
  + 专精的技术方向：测试技术方面
+ 项目面试（二面）
  + 价值
    + 为团队做了什么：质量方面、效率方面
    + 为项目做了什么：项目质量方面、项目效率方面、有没有推进项目
    + 为公司做了什么

+ 综合素质面试（三面）
  + 能力：
    + 团队协作能力
    + 团队管理能力
    + 核心竞争力
+ 精彩的自我介绍（面试的重点）
  + 深刻的第一印象
  + 优秀的语言表达能力：优秀的自我介绍
  + 亲和力
